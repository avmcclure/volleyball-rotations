## Project Overview

This is a TypeScript project for managing volleyball rotations. The project is in early development stages.

## Development Commands

### Build
```bash
npm run build
```
Compiles TypeScript files from `src/` to `dist/` directory.

### TypeScript Compilation
```bash
npx tsc
```
Direct TypeScript compilation (same as build script).

## Project Structure

- **src/**: TypeScript source files
- **dist/**: Compiled JavaScript output (generated by build)
- **tsconfig.json**: TypeScript configuration with strict mode enabled, targeting ES2016

## TypeScript Configuration

- Strict mode is enabled
- Target: ES2016
- Module system: CommonJS
- Output directory: dist/
